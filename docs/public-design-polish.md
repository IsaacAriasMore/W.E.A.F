# Pulido visual público

## Dirección

La zona pública conserva la identidad W.E.A.F: carbón mineral, superficies azul oscuro, Archivo Variable y ámbar fósil para acciones. El rediseño es una evolución de marca y no modifica rutas, contenidos legales ni la arquitectura de las zonas privada y admin.

## Superficies

- Header sticky con fondo translúcido, selector ES/EN y menú móvil.
- Home con hero de viewport completo, imagen original, herramientas públicas, privacidad de tribu, ASE/ASA, breeding, servidores, proceso, comunidad, FAQ y CTA final.
- Cards públicas con borde contenido, brillo interior y elevación breve.
- INIs, mapas, bosses, criaturas, servidores, publicación y legal comparten reveals progresivos.
- Footer con datos del creador, contacto y disclaimer de independencia.

## Límites

- No se añadió motion pesado a dashboard, breeding, admin, configuración o pagos.
- No se incorporaron librerías de animación.
- El contenido permanece visible cuando JavaScript no está disponible.
- La imagen hero existente tiene dimensiones reservadas para evitar saltos de layout.
